# 个人智能日程管理系统

一款基于 Python + Tkinter 开发的桌面端日程管理工具，集成了任务管理、习惯打卡、日历视图、天气同步、提醒设置、统计分析与个性化配置等功能，旨在帮助用户更高效地管理日常学习与生活安排。

---

## 项目简介

随着学习与生活节奏不断加快，如何高效安排任务、养成良好习惯、合理设置提醒，成为许多人日常管理中的核心需求。  
本项目围绕“轻量、直观、可扩展”的设计目标，开发了一款基于桌面端的智能日程管理系统，提供任务记录、状态切换、习惯追踪、提醒管理、天气信息同步和统计分析等能力。

项目采用 `Tkinter` 构建图形界面，配合本地 JSON 数据存储，实现了无需联网也可使用的核心日程管理功能。

---

## 功能特点

- **📋任务管理**
  - 新增、编辑、删除日程/任务
  - 支持任务完成状态切换
  - 支持任务备注记录
  - 支持任务颜色分类与可视化展示

- **📅习惯打卡**
  - 支持每日习惯记录
  - 可查看连续打卡情况
  - 便于培养学习、运动、早睡等长期习惯

- **🗓️日历与时间视图**
  - 可视化查看日程安排
  - 结合时间线展示每日任务

- **⏰提醒设置**
  - 支持设置提醒事项
  - 可开启/关闭提醒功能
  - 支持提前提醒时间配置

- **🌤️天气同步**
  - 支持配置城市
  - 可同步天气信息，便于结合天气安排外出或活动

- **📊统计分析**
  - 统计任务完成情况
  - 统计习惯打卡数据
  - 辅助用户了解自身执行效率

- **⚙️个性化设置**
  - 可修改昵称
  - 可设置头像
  - 可调整天气城市等配置项

- **💾本地数据保存**
  - 所有数据自动保存到本地 JSON 文件
  - 程序重启后数据仍可保留

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/6443ren/smart-schedule-manager.git
cd smart-schedule-manager
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

如果你希望手动安装，也可以使用：

```bash
pip install requests plyer pillow
```

### 3. 运行项目

```bash
python smart_schedule_manager.py
```

## 核心功能

### 任务管理
- 新增、编辑、删除任务
- 支持任务完成 / 未完成状态切换
- 选中任务后可自动回填编辑区
- 支持立即保存到本地 JSON

### 数据持久化
- 任务、打卡记录、设置统一保存
- 启动后自动加载历史数据
- 数据文件损坏时可自动重建

### 定时提醒
- 后台线程监听任务时间
- 支持提前提醒
- 支持系统通知、弹窗与提示音

### 习惯打卡
- 可添加自定义习惯
- 支持每日打卡
- 显示连续天数统计
- 提供圆点状态可视化

### 日历功能
- 月视图展示
- 任务日期标记
- 支持月份切换
- 点击日期可快速跳转

### 统计分析
- 任务完成率柱状图
- 任务分类饼图
- 使用 Tkinter Canvas 原生绘制

### 天气同步
- 接入 Open-Meteo 免费天气 API
- 无需 API Key
- 可结合天气辅助日程安排

## 数据说明

项目会在运行目录下自动生成以下数据文件：

```text
smart_schedule_data.json
```

该文件用于保存：

- 任务数据
- 打卡记录
- 用户设置
- 日程相关配置

如果你不希望提交该文件，请将其加入 `.gitignore`：

```gitignore
smart_schedule_data.json
```

## 使用说明

### 任务计划
进入任务页面后，可以添加日程、编辑内容、删除任务，并切换完成状态。

### 习惯管理
进入习惯打卡页面后，可以添加自己的习惯并进行每日记录，系统会自动统计连续打卡天数。

### 提醒设置
在提醒页面中可填写提醒内容、时间与提前提醒分钟数，程序运行时会自动检测并推送提醒。

### 天气同步
在天气页面中输入城市后即可同步天气信息，便于规划出行或安排活动。

### 统计分析
在统计页面中可以查看任务完成情况和分类统计，帮助你了解近期日程执行效果。

### 设置中心
在设置中心中可修改昵称、头像、天气城市等个性化信息。

## 常见问题

### 图片无法显示怎么办？
请检查：
- 图片是否已上传到 `assets/screenshots/`
- README 中的文件路径是否与实际文件名一致
- 中文文件名是否存在拼写或编码问题

### 程序提示缺少依赖怎么办？
执行以下命令即可：

```bash
pip install requests plyer pillow
```

### 数据会丢失吗？
不会。程序会自动将数据保存到本地 JSON 文件中。

### 是否支持多平台？
理论上支持 Windows、macOS 和 Linux，但部分系统通知效果可能略有差异。
